# Button

This component was created from Button of [@material-ui](https://material-ui.com).

[Button API](https://material-ui.com/api/button) (We replaced default color prop to 'primary' and prop variant to 'contained'.)

We also use the following additional props from [@material-ui/system](https://material-ui.com/system):

[Spacing API](https://material-ui.com/system/spacing)

[Sizing API](https://material-ui.com/system/sizing)

## Extra Props

| Name      | Description                                                                                                          | Default   |
| --------- | -------------------------------------------------------------------------------------------------------------------- | --------- |
| component | The component used for the root node. Either a string to use a HTML element or a component.                          | 'button'  |
| to        | Path to a new route (can only be used if the component is receving Link from react-router-dom in the component prop) | undefined |
