## Extra Props

| Name      | Description                                                                                                          | Default        |
| --------- | -------------------------------------------------------------------------------------------------------------------- | -------------- |
| component | The component used for the root node. Either a string to use a HTML element or a component.                          | 'button'       |
| to        | Path to a new route (can only be used if the component is receving Link from react-router-dom in the component prop) | undefined      |
| color     | The color of the component. This prop was replaced by the material-ui/system one.                                    | 'text.primary' |
