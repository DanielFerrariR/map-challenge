This component was created from modal of [@material-ui](https://material-ui.com).

[Modal API](https://material-ui.com/api/modal)

[Display API](https://material-ui.com/system/display)

<Preview>
  <Story name="simple modal">
    {() => {
      const [open, setOpen] = useState(false)
      return (
        <>
          <Button onClick={() => setOpen(true)} color="secondary">
            Open modal
          </Button>
          <Modal open={open} setOpen={setOpen}>
            <Typography mb={3} align="center">
              Modal Name
            </Typography>
            <Button
              color="secondary"
              size="large"
              width={0.5}
              onClick={() => setOpen(false)}
            >
              Ok
            </Button>
          </Modal>
        </>
      )
    }}
  </Story>
</Preview>

## Extra Props

| Name       | Description                                          | Default   |
| ---------- | ---------------------------------------------------- | --------- |
| open       | The state to open the modal.                         | undefined |
| setOpen    | The setState function to close the modal.            | undefined |
| paperProps | A parameter to change internal paper component props | undefined |
